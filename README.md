# KutezPage
 Asp.net project for Kutez Internship

 you need to update appsettings.json file in KutezApi part with your token

{
    "Logging": {
        "LogLevel": {
            "Default": "Information",
            "Microsoft.AspNetCore": "Warning"
        }
    },
    "AllowedHosts": "*",

    "GoldApi": {
        "ApiKey": "your api key",
        "Endpoint": "https://www.goldapi.io/api/XAU/USD"
    }
}
